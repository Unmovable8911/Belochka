## What to build

When no servers are configured, the dashboard shows a guided empty state instead of a blank grid. The empty state is a centered prompt with a clear call-to-action that opens the add-server dialog, making first-time setup obvious.

## Acceptance criteria

- [ ] Empty state displayed when server list is empty
- [ ] Centered layout with descriptive text (e.g., "No servers configured")
- [ ] Prominent "Add your first server" button
- [ ] Button opens the add-server dialog (from slice #5)
- [ ] Empty state disappears when first server is added
- [ ] Empty state reappears when last server is deleted
